What's new in

  • Franchise common files
    • Added support for editing .FTC files in the editor.
    • You can get these files from Frosty -> Legacy Editor -> common -> Franchise -> FranTkData -> 0
    • To enable your changes, you have to use Frosty to 'Import' your saved file from the editor. (Right click on the file -> Import)
    • Changes made to these files affect ALL franchises. To un-do your changes, you can use Frosty's 'Revert' option. (Right click on the file -> Revert)
