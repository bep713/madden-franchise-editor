What's new in v3.8.0

  • Update notifications
    • Added this window, showing important new features or things needing your attention.
  • Schema management
    • Added a schema management system that allows you to save custom schemas.
    • Added a service that allows you to scan the game files for updated schemas.
    • Added an alert message that shows when you open a file without an exact schema match saved.
  • Bug fixes
    • Fixed an issue where the ability editor did not include new abilities in the game.
    • Fixed an issue where the 'save has been modified' message appears after saving normally in the editor.
