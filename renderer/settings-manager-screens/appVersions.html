What's new in

  • Madden 22 support
    • Added base Madden 22 support. There may be some issues - please let me know! I don't actually have the game so I can't test things out.
  • Table editor - set empty reference
    • Added the ability to set an existing reference to empty (all 0s). To do this, edit the reference, click on the table dropdown and scroll all the way down to the bottom, there will be an option title 'no table - null value'. Click that and save the reference and it will be zeroed out.
